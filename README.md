# 📰 Fake News Detection System

## 📌 Project Overview
This project uses Machine Learning to classify news articles as Fake or Real.

## 🎯 Objective
To build a text classification model that can detect fake news using Natural Language Processing techniques.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP (TF-IDF)

## 📊 Dataset
The dataset contains real and fake news articles.
(Source: Kaggle Fake News Dataset)

## 🤖 Model Used
- Logistic Regression / Naive Bayes (Mee model edhi use chesavo adhi rayali)

## 📈 Model Accuracy
Accuracy: 94%  (Mee actual accuracy ento adhi pettu)

## ✨ Features
- Text preprocessing
- TF-IDF vectorization
- Model training
- Accuracy evaluation
- User input prediction

## ▶ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py

## 📌 Example
Input: "Aliens landed in Hyderabad"
Output: Fake News
